## 描述
基于 TypeScript 的坦克大战的非标准实现。

[**\>\>\> 开始游戏 <<<**](https://songquanpeng.github.io/battle-city/)

|按键|描述|
|---|---|
|F11|全屏并重新加载游戏|
|W|向上移动|
|A|向左移动|
|S|向右移动|
|D|向下移动|
|B|重新部署|
|空格键或者鼠标左键|开火|
|R|重新加载|
|C|清楚建筑物|
|P|暂停或继续游戏|
