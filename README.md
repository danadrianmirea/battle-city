## Hot key
|Key|Action|
|---|---|
|F11|Full screen; Note that doing this will reload the page|
|W|Move up|
|A|Move left|
|S|Move down|
|D|Move right|
|B|Rebirth|
|Blank|Fire|
|R|Reload the page|
