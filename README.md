# Battle City

[**\>\>\> Start Game <<<**](https://songquanpeng.github.io/battle-city/)

## Instructions
|Key|Action|
|---|---|
|F11|Enter fullscreen mode and reload the page|
|W|Move up|
|A|Move left|
|S|Move down|
|D|Move right|
|B|Rebirth|
|Space bar / Left mouse key|Fire|
|R|Reload the page|
|C|Remove all buildings|
|P|Pause or resume game.|
